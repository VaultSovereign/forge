Quickstart

1) Install toolchain

# Node 22 and pnpm
corepack enable
corepack prepare pnpm@10.17.0 --activate

2) Install deps and build

pnpm install --frozen-lockfile
pnpm run build

3) Run tests

pnpm test             # unit tests
pnpm run test:forge  # optional Python-based deterministic checks

4) Run Workbench

Dev (fast refresh):

pnpm run dev

Prod-like single port:

pnpm run build
pnpm run workbench:start

5) Common scripts

  • pnpm run typecheck — TypeScript typecheck
  • pnpm run lint — Lint
  • make forge-prepush — Local CI gate (sets FORGE_SKIP_REMOTE_SCANS=1 if no provider)
