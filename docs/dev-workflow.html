Dev Workflow

Typical Loop

  1. Edit code
  2. pnpm run build or pnpm run dev
  3. pnpm test
  4. Commit with Conventional Commits
  5. Open PR (CI runs)
  6. Merge
  7. Docs auto-publish to GitHub Pages

Commands

pnpm run typecheck
pnpm run lint
pnpm test
make forge-prepush  # local CI mirror (skips remote scans without provider)

Docs

  • Docs site: GitHub Pages
  • Edit files in docs/ and commit; Pages publishes automatically
