Forge Basics

Quick orientation for working with the Forge repository.

Requirements

  • Node 22.x
  • pnpm 10.17+
  • Python 3.10+ (only for optional tests and tools)

Install

pnpm install --frozen-lockfile
pnpm run build
pnpm test

Run the Workbench

Development (two processes, hot reload):

pnpm run dev

Production-like single port:

pnpm run build
pnpm run workbench:start

Repo Layout (very short)

  • agents/ Guardian/runner agents
  • agent/ planner/executor library (CLI/MCP)
  • dispatcher/ prompt expansion + validation
  • workbench/ BFF + frontend
  • catalog/ templates; schemas/ JSON schemas
  • docs/ docs site (this)
